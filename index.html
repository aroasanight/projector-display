Projector Display

I've spent quite a lot of time on creating this display and all the information/websites surrounding it - I hope you find it as interesting as I did! :D

This display is an abstraction of a projector. We've removed all structural components, all casings (apart from the main shell so you can still easily tell what it is), and the majority of the wires. We've then reconstructed all the parts together again, but vertically separated - so you can see all the individual components.

If you're wondering, as of the last time this page was updated (sep 21 at 19:41), I've spent 9 hours and 21 minutes working on this.
